
# 1주차 미션- 숫자야구게임 만들기!-
---------------------------------
### 구현할 기능 목록

* 랜덤으로 서로 다르고 0이 아닌 3자리 숫자 만들기
    * 랜덤으로 0과 9사이의 숫자를 만든다.
    * 0이 아닌지 확인한다.
    * 기존에 뽑은 숫자와 동일한 숫자를 뽑았는지 확인한다.

* 사용자의 입력이 올바른 입력인지 판별하기
    * 입력의 길이가 3인지 확인한다.
    * 모두 숫자인지 확인한다.
    * 서로 모두 다른 숫자인지 확인한다.
    * 0이 없는지 확인한다.

* 결과 계산하기
    * 주어진 index가 strike, ball인지 확인한다
    * 깔끔하게 스트라이크, 볼, 낫싱을 String으로 출력한다.

* 정답을 맞췄는지 확인하기

  (정답이 맞을 경우)
    * 태그(정답을 맞추셨습니다!) 추가하기.
    * 다시하기 버튼 생성하기.
    * 다시 시작하는 restart 함수 만들기
        * 입력창 비우기
        * 추가된 버튼과 HTML를 모두 원상복귀 시킨다.
    * 다시하기 버튼에 restart 이벤트 리스너 추가하기

  (정답이 아닐 경우)
    * 결과 출력하기

* 확인버튼에 게임을 진행할 수 있도록 이벤트리스너 추가하기
